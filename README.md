# Interactive Field Guide · Claude Skill

A Claude skill that produces polished, interactive HTML "field guides" — multi-section research reports with sidebar navigation, search (⌘K), glossary (⌘G), dark mode, click-to-expand drawers, an SVG constellation map of an ecosystem, a 2×2 strategic matrix, paired-bar charts, tab-switched case studies, and a 12-part editorial structure.

Use it for industry analysis, competitive landscapes, ecosystem strategy maps, fund theses, market deep-dives, and founder research memos — anything that should feel more substantive than a markdown report and more visually-cohesive than ad-hoc HTML.

## Install

1. Download the latest release ZIP, or build from source:
```bash
2. In Claude.ai → **Settings → Capabilities** → enable **Code execution and file creation**
3. Go to **Customize → Skills** → click **+** → **Create skill** → **Upload a skill**
4. Select the ZIP file
5. Toggle the skill on

Detailed Claude.ai install guide: https://support.claude.com/en/articles/12512180

## How to trigger

Once installed, the skill triggers when you ask Claude things like:

- "Make me a field guide on the [industry] ecosystem"
- "Deep dive on [Company] — how do they make money, who are the competitors, what's the recent strategy"
- "Build an interactive research report on [topic]"
- "Map [Company]'s strategic ecosystem — every partner, investor, and acquirer in the last 12 months"
- "Competitive landscape of [vertical]"
- "Fund thesis on [theme] with company evidence and an actionable judgments table"
